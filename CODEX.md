# BrowserPane Codex Guide

This repo keeps its canonical project instructions in `AGENTS.md`.

Read `AGENTS.md` first and treat it as the source of truth for:
- current support matrix
- architecture map
- protocol and media assumptions
- validation commands

This file is intentionally thin. Update `AGENTS.md` first, then adjust `CODEX.md` only if the Codex entrypoint needs to change.

OpenAI's Codex guidance refers to `AGENTS.md` as the native persistent repo-instructions file. `CODEX.md` is a compatibility wrapper that points back to it.
